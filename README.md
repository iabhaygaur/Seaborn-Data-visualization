# Exploratory Data Analysis (EDA) & Data Visualization Dashboard

A comprehensive data analysis and visualization repository demonstrating end-to-end Exploratory Data Analysis (EDA) using Python. This project evaluates multi-domain datasets—specifically covering retail superstore performance metrics and student institutional demographics—to extract actionable business insights and trends.

---

## 🛠️ Tech Stack & Libraries
* **Language:** Python
* **Libraries:** Pandas, Seaborn, Matplotlib, NumPy
* **Environment:** Jupyter Notebook / Anaconda

---

## 📊 Core Analytics & Insights Included

### 1. Retail Superstore Performance Analysis
* **Regional Distribution:** Analyzed sales across core geographical zones (Central, East, West, South) to identify top-performing business markets.
* **Segment Breakdown:** Visualized product category distributions and order volumes across distinct customer segments (Consumer, Corporate, Home Office).
* **Temporal Trend Analysis:** Converted and structured transactional timestamps to plot year-over-year sales trajectories across primary divisions.
* **Profitability Mapping:** Implemented multi-variable scatter plots to map raw sales volume against net profit margins, isolating high-value transaction clusters.

### 2. Student Enrollment & Demographic Insights
* **Demographic Spreads:** Utilized detailed histograms with overlaid Kernel Density Estimate (KDE) curves to track audience age distributions segmented by gender.
* **Statistical Distribution:** Employed horizontal box plots to evaluate median age ranges, variance, and isolate statistical outliers.
* **Automated Domain Extraction:** Replaced conventional loops with optimized vectorized string methods in Pandas to extract, clean, and visualize target email provider domains.
